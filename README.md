# Локальная AI-инфраструктура: Llama3 в WSL2



## Описание проекта 
Этот проект демонстрирует настройку полноценной инженерной среды для запуска больших языковых моделей (LLM) локально. Основная цель - обеспечить приватность данных и независимость от внешних облачных API (OpenAI и др.).

Что я реализовала (My Tech Stack) 
* **Окружение:** Windows 11 + WSL2 (Ubuntu 22.04 LTS).
* **AI Engine:** Ollama.
* **Модель:** Llana 3 (8B параметров).
* **Инструментарий:** Bash, конфигурация гипервизора (INI).

## Решенные инженерные задачи 
### 1. Управление ресурсами (Resourse Allocation)
Столкнулась с ограничением RAM в виртуальной среде.
* **Проблема** Модель требовала больше памяти, чем выделял WSL2 по умолчанию.
* **Решение** Сконфигурировала файл ".wslconfig' в корневом каталоге хоста, расширив пул оперативной памяти до 8GB
* **Команда проверки** 'free -h'.

  ### 2. Деплой и инфференс
  * Развернула движок Ollama внутри Linux-подсистемы.
  * Провела первый инференс (запуск модели) и настроила взаимодействие на русском языке.
